IF ( a ? b : c )

a = 1
b = 20

if b > a:   
# IF!!! b is MORE THAN a, THEN do the following
  print("b is greater than a")

# ELSE!!! b is LESS THAN a, THEN do the following
else:
  print("b is less than a")

# Prints: b is greater than a

Quite simple, it’s also know that philosophers use logic to deduct conclusions from their metaphysics in a philosophy. The exact logic we communicate in philosophy is the same as that of programming. Thus, PROGRAMMING IS A PHILOSOPHY

IF !!! we ask whether A is *true/false* then we receive an answer that:

  1. B should be done if A is *true*
  2. C should be done if A is *false*
  3. Therefore A ? B : C is the “Ternary Operand”…

ELSE IF (a ? b : (c ? d : e))

a = 1
b = 20

if b > a:   
# IF!!! b is MORE THAN a, THEN do the following
  print("b is greater than a")

elif b == a:
# ELSE IF!!! b is EQUAL TO a, THEN do the following    
  print("b is equal to a")

else:
# ELSE!!! b is LESS THAN a, THEN do the following
  print("b is less than a")

# Prints: b is greater than a

IF !!! we ask whether A is *true/false* then we receive an answer that:

  1. IF A is *true* then we need to do B.
  2. IF A is *false* then we receive an answer that C is *true/false*.
  3. IF C is *true* then we do D.
  4. IF C is *false* then we do E.
